# Markdown Cheatsheet

## Headings

```md
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

## Text Formatting

```md 
**in bold**

*in italics*

`code`

> block quote

Fenced code block
    ```js
    {
    "firstName": "John",
    "lastName": "Smith",
    "age": 25
    }
    ```

```

**in bold**

*in italics*

`code`

> this is an block quote

fenced code block
```js
{
    "firstName": "John",
    "lastName": "Smith",
    "age": 25
}
```

## Links & Images
